# Trabalho de frontend para ChatGPT da Universidade

Utilizando apenas a stack de Javascript, html e css foi construída a interface de um chatbot que ajudará os alunos da universidade com questões da prórpia instituição.
O site deveria ser dinâmico e aceitar tanto para smartphone quanto para computador. Além disso, o site foi hospedado em https://chatgpt-ufop.vercel.app para mais fácil visualização.
Ademais, é possível rodar um servidor mockado seguindo as instruções:
- node server.js
- http://localhost:2200/TelaInicial.html

Tivemos que nos atentar a não deixar o usuário mandar mensagem enquanto a última carrega, fazer o input aumentar até certo número de linhas e criação de animação no loading da mensagem.
